Albert Einstein – Interactive Biography Website

An interactive, single-page biography website dedicated to Albert Einstein, built using HTML5, CSS, and JavaScript. The project presents Einstein’s life, discoveries, and publications through a structured layout, smooth navigation, and interactive UI elements.

The website features a detailed About section covering Einstein’s life and career, an interactive Discoveries section with expandable cards explaining his major scientific contributions, and a Publications section showcasing his most influential research papers with direct links. A gallery section visually highlights key moments from different stages of his life, while Fun Facts and Blogs sections add engaging and extensible content.
